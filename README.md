# Servicio b

Implementación de un servicio simple que depende de los datos de una API externa

## Requerimientos

async 2.6.3

express 4.14.0

request 2.88.2

## Uso

node index.js

Posteriormente en el navegador se puede acceder a localhost:8010/numbers

Devuelve un JSON obtenido de una API externa
